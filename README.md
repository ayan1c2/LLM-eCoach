# Supplementary Materials

This repository includes four supplementary files that support the proposed Health eCoach system. Each item is briefly described below.

## Contents

- **Supplementary Material 1 — Semantic OWL Ontology Structure**  
  Structured representation of the key entities and relationships in the proposed semantic OWL ontology. This file enables effective knowledge representation and reasoning about user behaviours, preferences, and goals.

- **Supplementary Material 2 — Sample SPARQL Queries**  
  Example SPARQL queries used within the eCoaching system to retrieve ontology-grounded information for personalized recommendation generation.

- **Supplementary Material 3 — LLM Question & Conversation Samples**  
  - **A.3.1: Probable User Questions**  
    Sample questions that may be asked by users and handled by the LLM-based eCoach.  
  - **A.3.2: Week-long Conversation Examples**  
    Sample probable conversations (text prompts to ChatGPT) across key behavioural categories—activity, sleep, diet, and habit—illustrating personalized behaviour optimization and goal management over one week.

- **Supplementary Material 4 — Sample Chatbot Interactions**  
  Example end-to-end chatbot dialogues showing how the prototype responds to user inputs and generates sentiment-aware, ontology-grounded recommendations.
